# AI Agent 开发指南

## 开发环境设置

### 必要条件
- Node.js >= 16
- Yarn 包管理器
- Ollama 本地服务
- Git 版本控制

### 环境配置步骤
1. 克隆项目
```bash
git clone [repository-url]
cd ai-agent
```

2. 安装依赖
```bash
yarn install
```

3. 启动开发服务器
```bash
yarn dev
```
